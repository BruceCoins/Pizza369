# Truffle 框架智能合约测试    
- 智能合约测试的两种方法：  
  1> 区块链级别的 Solidity 测试。  
  2> DApp级别的 JavaScript 测试。
    
- 智能合约测试的类型主要有三种：  
  1> 单元测试：测试合约的单个函数或功能  
  2> 集成测试：测试合约间的交互  
  3> 系统测试：测试整个dapp（去中心化应用）或系统
  
- JavaScript 测试主要使用 Mocha 和 Chai 断言 编写测试脚本。
